Beskrivning

Applikationen läser in information om husbilar som kan hyras ut. Man kan filtrera på olika typer av bilar. Om man hyr ut själv kan man logga in och se vyer anpassade för de egna bilar man har. Byggd med React Router 6 som hanterar client-side routing.

Description

Web application where you can filter out desired van type and see descriptions and prices. If you login you can see views designed for your own Vans.

This app is built with React Router 6 to handle client-side routing.
